Install guide

Simple setup. Fast start.

Use the latest build for your version, place it in your mods folder, and make sure the recommended dependencies are installed.

1

Download the correct version

Pick the release that matches your Minecraft setup. The addon supports multiple 1.21.x versions, so choose the build that lines up with your client.

2

Drop the jar into your mods folder

After downloading, move the jar file into your Minecraft mods folder so Fabric and Meteor can load it on startup.

3

Install recommended dependencies

For the best compatibility, use XaeroPlus, Xaero’s Minimap, Xaero’s World Map, and a Meteor-friendly Baritone build.

Some features may not appear if dependencies are missing.
4

Launch and configure

Open Meteor, enable the modules you want, and set webhook options or movement settings to fit your own workflow.

Avoid switching Search Area modes while it is running, since that is listed as a known crash source.
FAQ

Quick answers before you download.

Based on the repository notes and common setup questions.

Where do I get the jar?

Download the latest release for your version and place it in your mods folder. This page is designed to drive users to those release downloads fast.

Why is a module missing?

Usually because a dependency is missing. The repo recommends XaeroPlus, Xaero’s Minimap, Xaero’s World Map, and Baritone for full functionality.

Why is Search Area moving in a straight line?

It saves your path and can continue where you left off. To start a fresh route, change the name or use the reset option.

What if the game crashes?

A known crash can happen if you switch Search Area modes while using it. The README also notes potential issues when using PathSeeker alongside the addon.
